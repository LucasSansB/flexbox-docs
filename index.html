Flexbox

Documentação

display: flex

A primeira propriedade que precisa ser declarada para que o Flexbox funciona

Algumas propriedades com flex-direction, justify-content e align-content já vem com os valores padrão.

Propriedades Valor Padrão
flex-direction row
justify-content flex-start
align-content flex-start

Exemplos sobre ''

row | row-reverse | column | column-reverse | initial | inherit

.container {
    display: flex;
    flex-direction: row;
    justify-content: flex-start;
    align-content: flex-start
}
